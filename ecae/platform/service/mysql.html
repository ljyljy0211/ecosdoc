MySQL

服务概述

ECAE平台提供的数据库服务。默认是开启的，您可以通过phpmyadmin来创建数据库和数据表。

您可以使用ECAE预定义常量来连接数据库:

用 户 名 ECAE_MYSQL_USER
密　　码 ECAE_MYSQL_PASS
主库域名 ECAE_MYSQL_HOST_M
从库域名 ECAE_MYSQL_HOST_S
数据库名 ECAE_MYSQL_DB

使用指南

对表的一些操作实例

<?php
    # 连接数据库
    $db = ECAE_MYSQL_DB;
    $user =  ECAE_MYSQL_USER;
    $pass =  ECAE_MYSQL_PASS;
    $host =  ECAE_MYSQL_HOST_M;
    $lnk = mysql_connect($host,$user,$pass);
    mysql_select_db($db,$lnk);

    # 创建表
    $sql = 'create table test(id int(10) primary key auto_increment,name varchar(20) default "")';
    $r = mysql_query($sql,$lnk);
    echo "add table:{$sql}<br>";
    var_dump($r);
    echo "<HR>";

    # 插入数据
    $sql = 'insert into test( `id`,`name` ) values ('.time().','.time().')';
    echo "insert table test:{$sql}<br>";
    $r = mysql_query($sql,$lnk);
    var_dump($r);
    echo "<HR>";

    # 查看数据
    $sql = 'select * from test';
    echo "select db test:$sql<br>";
    $r = mysql_query($sql,$lnk);
    while($re = mysql_fetch_assoc($r)){
        print_r($re);
        echo "<br>";
    }
    echo "<HR>";

    # 删除表
    $sql = 'drop table test';
    echo "drop table: {$sql}<br>";
    $r = mysql_query($sql,$lnk);
    var_dump($r);
    echo "<HR>";
