iframeedit 订单更新

返回参数名称 类型 是否必须 描述
tid String y 交易编号（交易id）
title String n 交易标题
created Date n 交易创建时间。格式:yyyy-MM-dd HH:mm:ss
modified Date n 交易修改时间。格式:yyyy-MM-dd HH:mm:ss
status String n 交易状态，可选值： TRADE_ACTIVE:交易处理中, TRADE_CLOSED:交易作废即交易关闭, TRADE_FINISHED:交易成功
pay_status String n 交易支付状态，可选值： PAY_NO:未付款, PAY_FINISH:已付款, PAY_TO_MEDIUM:付款至担保交易, PAY_PART:部分付款, REFUND_PART:部分退款, REFUND_ALL:全额退款
ship_status String n 交易物流状态，可选值： SHIP_NO:未发货, SHIP_PREPARE:配货中, SHIP_PART:部分发货, SHIP_FINISH:全部发货, RESHIP_PART:部分退货, RESHIP_ALL:全部退货
has_invoice Boolean n 是否开发票。可选值：true(开),false(不开)。默认不开
invoice_title String n 发票title。如果要invoice为true,则此参数为必须
invoice_fee String n 发票税金
total_goods_fee String n 商品总额(不包括配送费用)
total_trade_fee String n 交易应付总额（包括商品价格 + 配送费用)
discount_fee String n 折扣优惠金额
goods_discount_fee String n 商品优惠金额
orders_discount_fee String n 订单优惠金额
promotion_details PromotionDetail n 详细优惠方案 json{ “promotion_name string 优惠描述”, “promotion_fee String 优惠金额”, “pmt_id String 优惠对应的订单ID，可能是交易号tid,也可能是oid.ID决定优惠是在交易上还是子订单上” }
payed_fee String n 已支付金额
Currency String n 当前交易选择的货币类型。可选值：CNY（人民币），USD（美元），JPY（日元）。- 默认为CNY（人民币）
currency_rate String n 当前货别汇率
total_currency_fee String n 当前货币订单总额
buyer_obtain_point_fee String n 买家获得积分,返点的积分。格式:100;单位:个
point_fee String n 买家使用积分。格式:100;单位:个
total_weight Number n 该笔交易的商品总重量
receiver_time String n 收货人要求到货时间
shipping_tid String n 创建交易时的物流方式ID（交易完成前，物流方式有可改变）
shipping_type String n 交易物流方式名称
shipping_fee String n 交易物流配送费用
is_protect String n 是否保价（针对物流配送)
protect_fee String n 保价费用（针对物流配送)
payment_tid Number n 支付方式ID
payment_type String n 支付方式名称
is_cod Boolean n 是否货到付款。可选值:true(是),false(否)
receiver_name String n 收货人的姓名
receiver_email String n 收货人email地址
receiver_mobile String n 收货人的手机号码
receiver_state String n 收货人的所在省份
receiver_city String n 收货人的所在城市
receiver_district String n 收货人的所在地区
receiver_address String n 收货人的详细地址
receiver_zip String n 收货人的邮编
receiver_phone String n 收货人的电话号码
pay_cost String n 支付手续费
buyer_memo String n 买家备注
orders_number Number n 当前交易下订单数量
buyer_uname String n 买家帐号
buyer_name String n 买家姓名
buyer_address String n 买家的详细地址
buyer_mobile String n 买家的手机号码
buyer_phone String n 买家的电话号码
buyer_email String n 买家email
buyer_zip String n 买家的邮编
Orders Order n 订单列表
Payments Payment n 支付单列表
