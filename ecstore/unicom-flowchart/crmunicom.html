ecstore 联通 crm 业务

新建订单


  1. ecstore创建订单；
  2. 通过接口（creatorder）抛数据给crm ；
  3. crm根据接收的订单数据创建订单
  4. 通过callback地址，返回创建订单信息

订单支付


  1. ecstore/ocs/erp发起支付动作；
  2. 订单支付状态变更；
  3. 通过接口（orderpay）接口抛订单数据给crm；
  4. crm接收数据处理完成，callback数据给ecstore

订单发货


  1. ecstore ocs erp 发起发货动作；
  2. 订单发货状态变更；
  3. 通过接口（orderupdate）接口抛订单数据给crm；
  4. crm接收数据处理完成后callback处理结果给ecstore

订单退款


  1. ecstore ocs erp 发起退款动作；
  2. 订单支付状态变更；
  3. 通过接口（orderupdate）接口抛订单数据给crm；
  4. crm接收数据处理完成后callback处理结果给ecstore

订单退货


  1. ecstore ocs erp 发起退款动作；
  2. 订单发货状态变更；
  3. 通过接口（orderupdate）接口抛订单数据给crm；
  4. crm接收数据处理完成后callback处理结果给ecstore

订单取消


  1. ecstore ocs erp 发起订单取消动作；
  2. 订单状态变更；
  3. 通过接口（orderupdate）接口抛订单数据给crm；
  4. crm接收数据处理完成后callback处理结果给ecstore
