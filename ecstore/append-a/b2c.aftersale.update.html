b2c.aftersale.update 修改售后申请

API应用级输入参数

参数 类型 是否必须 说明
order_bn string Y 订单号
return_bn string Y 退货记录流水号标识
title string N 售后服务标题
content string N 退货内容
comment string N 管理员备注
currency string N 退货记录状态
status bool N 状态
member_id string N 申请人
add_time string N 售后处理时间
url string N
return_product_items string N 商品详情

return_product_items为json格式，具体参数如下：

参数 类型 是否必须 说明
bn string y 货号
num int y 数量
name string y 商品名称

status 状态取值

参数 类型
1 申请中
2 审核中
3 接受申请
4 完成
5 拒绝
6 已收货
7 已质检
8 补差价
9 已拒绝退款

常见错误：

  1. 售后服务单数据异常！ 缺少order_bn return_bn参数
