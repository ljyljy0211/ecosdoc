web server相关配置

$_SERVER['PATH_INFO']

ecstore框架中，路由参数解析默认用的是

$_SERVER['PATH_INFO']
所以必需开启web server的pathinfo才能正常使用ecstore。而常用的nginx默认是没有开启的。

请参照此篇文章开启nginx下的pathinfo.
