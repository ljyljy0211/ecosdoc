ECStore1.2 安装search( sphinx )

1. install sphinx

http://sphinxsearch.com

2. install swcs

www.ftphp.com/scws/

3. install app

svn checkout http://app.ec-os.net/svn/search/branches/dev

svn checkout  http://app.ec-os.net/svn/scws/branches/dev

svn checkout http://app.ec-os.net/svn/sphinx/branches/dev

4.sphinx.conf

/path/to/app/sphinx/sphinx.conf

5. build index

sudo /path/to/sphinx/bin/indexer --config /path/to/app/sphinx/sphinx.conf --all

6. start searchd

sudo /path/to/sphinx/bin/searchd -c /path/to/app/sphinx/etc/sphinx.conf

that's all !
