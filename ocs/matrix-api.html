matrix对接接口说明

概述

本文档旨在说明OCS系统在和矩阵的交互过程中，访问了到了哪些矩阵接口，矩阵接口的详细描述可以在给出的在线文档中查询Product operating manual开放平台API文档地址

OSC使用到的开放平台接口

商品类

  • store.items.quantity.list.update 库存批量更新

店铺类

  • store.shop.payment_type.list.get 获取店铺支付方式列表

交易类

  • store.trade.aftersale.add 添加一笔交易的售后申请
  • store.trade.aftersale.status.update 修改一笔交易售后申请状态
  • store.trade.buyer_message.add 添加一笔交易的买家留言
  • store.trade.memo.add 添加一笔交易备注
  • store.trade.memo.update 修改一笔交易备注
  • store.trade.payment.add 添加交易付款单
  • store.trade.refund.add 添加交易退款单
  • store.trade.reship.add 添加交易退货单
  • store.trade.shipping.add 添加交易发货单
  • store.trade.shippingaddress.update 修改一笔交易的收货地址
  • store.trade.status.update 更新交易状态
  • store.trade.update 修改单笔交易信息
  • store.trade.reship.status.update 更新交易退货单状态
  • store.trade.ship_status.update 更新交易发货状态
  • store.trade.shipping.status.update 更新交易发货单状态
  • store.trade.shipping.update 修改交易发货单
  • store.trade.delivery.send 交易发货处理
  • store.trade.pay_status.update 更新交易支付状态
  • store.trade.payment.status.update 更新交易支付单状态
  • store.trade.refund.status.update 更新交易退款单状态

物流类

  • store.wms.item.add 添加商品
  • store.wms.item.update 修改商品
  • store.wms.item.query 根据商家商品编码查询商品信息
  • store.wms.inorder.create 创建入库单
  • store.wms.inorder.cancel 撤销入库单
  • store.wms.outorder.create 创建出库单
  • store.wms.outorder.cancel 撤销出库单
  • store.wms.saleorder.create 创建发货单
  • store.wms.saleorder.cancel 撤销发货单
  • store.wms.returnorder.create 创建退货单
  • store.wms.returnorder.cancel 撤销退货单
